# Corelytics
A Corelytics alkalmazás segítségével a testedzés és tréning hatékonyabbá és átláthatóbbá válik. Az app egyszerű eszközöket kínál a teljesítmény nyomon követésére és az edzésterv optimalizálására. Így a felhasználó könnyedén érheti el a kitűzött sportcéljait.
